# インストール

xlkitは、npm、pnpm、yarnのいずれかでインストールできます。

## パッケージマネージャーでインストール

```bash
npm install xlkit
```

```bash
pnpm add xlkit
```

```bash
yarn add xlkit
```

## 動作環境

- Node.js 18.0以上
- ブラウザ（モダンブラウザをサポート）

## TypeScript

xlkitはTypeScriptで書かれており、型定義が同梱されています。追加の型定義パッケージは不要です。
