# インストール

xlmakeは、npm、pnpm、yarnのいずれかでインストールできます。

## パッケージマネージャーでインストール

```bash
npm install xlmake
```

```bash
pnpm add xlmake
```

```bash
yarn add xlmake
```

## 動作環境

- Node.js 18.0以上
- ブラウザ（モダンブラウザをサポート）

## TypeScript

xlmakeはTypeScriptで書かれており、型定義が同梱されています。追加の型定義パッケージは不要です。
